NumPy(Numerical Python) 是 Python 语言的一个扩展程序库，支持大量的维度数组与矩阵运算，此外也针对数组运算提供大量的数学函数库。

NumPy 的前身 Numeric 最早是由 Jim Hugunin 与其它协作者共同开发，2005 年，Travis Oliphant 在 Numeric 中结合了另一个同性质的程序库 Numarray 的特色，并加入了其它扩展而开发了 NumPy。NumPy 为开放源代码并且由许多协作者共同维护开发。

间隔数组创建

等差数据数组：start为起始元素，stop为末尾元素（不包括），step为步长（即相邻数据的差值）

等间隔数据数组：以start为起点stop为终点画一条直线，num为该直线上的点的个数，且相邻点之间的距离要相等。例如（1, 11 ,3)，print(arr)的输出为：1，6，11。

对数间隔数据数组：先进行linspace计算，然后再进行以base为底数的幂运算。例如（1, 11, 3, base=2)，linspace的计算结果为1，6，11，最终logspace的输出结果为：2^1, 2^6, 2^11。

ndarray的运算

对于加减乘除，直接相应的元素进行运算；对于数组和标量的运算，则是数组中的每个元素都和标量进行运算

对于不同shape的数组的运算，采用广播机制。

如果需要对数组进行矩阵运算，用@符号：a @ b
